# Melody Trip - 간소화 IA (3시간 MVP)

## 1. 사이트 맵

### 1.1 전체 사이트 구조

```
Melody Trip (Simple)
│
├─ 대시보드 (/)
│  ├─ 🎵 음악 장르별 통계 (카드 또는 차트)
│  ├─ 🌍 대륙별 여행 통계 (카드) ⭐ 새로 추가
│  ├─ ✈️ 여행 카테고리별 통계 (카드) ⭐ 새로 추가
│  ├─ 📊 기본 통계 (총 여행, 국가 수, 음악 수)
│  ├─ 최근 여행 목록 (음악 정보 + 대륙 + 카테고리 포함)
│  └─ [+ 새 여행 추가] 버튼
│
├─ 새 여행 추가 (/add)
│  └─ 폼 (여행지, 국가, 대륙, 카테고리, 날짜, 좌표, 🎵 음악 정보, 메모)
│
├─ 여행 편집 (/edit/:id)
│  └─ 편집 폼 (기존 데이터 미리 채워짐)
│
├─ 세계 지도 (/map) ⭐ 추가
│  ├─ 🗺️ 2D 인터랙티브 지도
│  ├─ 여행지 마커 (장르별 색상)
│  └─ 마커 클릭 → 여행 정보 팝업
│
└─ 챗봇 (/chatbot)
   ├─ 채팅 인터페이스
   └─ 🎵 음악 기반 여행지 추천
```

## 2. 사용자 흐름

### 2.1 여행 + 음악 기록 생성 ⭐ 핵심

```
[대시보드]
    ↓
[+ 새 여행 추가 클릭]
    ↓
[새 여행 추가 페이지] (/add)
    │
    ├─ 여행지 입력 (예: "Paris")
    ├─ 국가 입력 (예: "France")
    ├─ 날짜 선택
    ├─ 🎵 음악 정보 입력
    │  ├─ 곡명 (예: "La vie en rose")
    │  ├─ 아티스트 (예: "Édith Piaf")
    │  └─ 장르 선택 (드롭다운: 팝, 인디, 클래식, 록, 재즈, 힙합, 기타)
    └─ 메모 입력 (선택)
    ↓
[저장 버튼 클릭]
    ↓
[Supabase에 저장]
├─ 1. songs 테이블에 음악 저장
└─ 2. trips 테이블에 여행 저장 (song_id 참조)
    ↓
[대시보드로 리다이렉트]
    ↓
[새 여행 카드 표시]
├─ 여행지, 국가, 날짜
├─ 🎵 음악 정보 (곡명, 아티스트, 장르)
└─ 장르별 색상 뱃지
```

### 2.2 음악 기반 챗봇 사용 ⭐ 핵심

```
[대시보드]
    ↓
[챗봇 버튼 클릭]
    ↓
[챗봇 페이지] (/chatbot)
    ↓
[질문 입력]
예: "잔잔한 인디 음악을 좋아해. 어디로 여행가면 좋을까?"
또는: "La vie en rose 같은 음악 좋아하는데 추천해줘"
    ↓
[전송 버튼 클릭]
    ↓
[LLM API 호출]
├─ 시스템 프롬프트: "음악과 여행 연결 전문가"
├─ 사용자 질문
└─ 컨텍스트: 사용자의 과거 여행 기록 + 음악 장르 데이터
    ↓
[챗봇 응답]
├─ 추천 여행지
├─ 음악과의 연관성 설명
└─ 💡 과거 기록 참고 (예: "당신은 인디 음악과 함께 도쿄를 다녀오셨네요!")
```

### 2.3 대시보드에서 장르별 통계 확인

```
[대시보드]
    ↓
[음악 장르별 통계 섹션]
    │
    ├─ 인디: 5회 🎸 (파란색)
    ├─ 팝: 3회 🎤 (빨간색)
    ├─ 재즈: 2회 🎺 (녹색)
    └─ ...
    ↓
[패턴 인식]
"아, 나는 인디 음악과 함께 여행을 가장 많이 다니는구나!"
```

## 6. 페이지별 콘텐츠

### 6.1 대시보드 (/) ⭐ 핵심 (강화!)

```
├─ 페이지 제목: "🎵 Melody Trip"
├─ 음악 장르별 통계 섹션 ⭐
│  ├─ 카드 또는 간단한 막대 차트
│  ├─ 각 장르별 여행 횟수
│  └─ 장르별 색상 구분
│     예: 인디 🎸 5회 (파란색)
│          팝 🎤 3회 (빨간색)
│          재즈 🎺 2회 (녹색)
├─ 대륙별 여행 통계 섹션 ⭐ 새로 추가
│  ├─ 카드 형태
│  ├─ 각 대륙별 여행 횟수
│  └─ 대륙별 방문 국가 수
│     예: 유럽 🇪🇺 4회 (3개국)
│          아시아 🌏 3회 (2개국)
│          북미 🌎 2회 (1개국)
├─ 여행 카테고리별 통계 섹션 ⭐ 새로 추가
│  ├─ 카드 형태
│  └─ 각 카테고리별 여행 횟수
│     예: 도시 🏙️ 5회
│          문화 🏛️ 3회
│          자연 🏔️ 1회
├─ 기본 통계 카드
│  ├─ 총 여행 횟수
│  ├─ 방문 국가 수
│  └─ 저장된 음악 수
├─ 액션 버튼
│  ├─ + 새 여행 추가
│  └─ 챗봇 열기
└─ 최근 여행 목록 (음악 + 대륙 + 카테고리 정보 포함) ⭐
   └─ 각 카드:
      ├─ 여행지, 국가, 대륙, 카테고리, 날짜
      ├─ 🎵 음악 정보 (곡명, 아티스트, 장르)
      ├─ 장르별 색상 뱃지
      ├─ 메모
      └─ [편집] [삭제] 버튼
```

### 6.2 새 여행 추가 (/add) ⭐ 핵심 (강화!)

```
├─ 페이지 제목: "새 여행 추가"
├─ 폼 필드
│  ├─ 여행지 * (텍스트)
│  ├─ 국가 * (텍스트)
│  ├─ 대륙 * (드롭다운) ⭐ 새로 추가
│  │  옵션: 아시아, 유럽, 북미, 남미, 아프리카, 오세아니아
│  ├─ 여행 카테고리 * (드롭다운) ⭐ 새로 추가
│  │  옵션: 도시, 자연, 해변, 문화, 모험, 휴양
│  ├─ 날짜 * (date picker)
│  ├─ 좌표 (선택사항)
│  │  ├─ 위도 (숫자)
│  │  └─ 경도 (숫자)
│  ├─ 🎵 음악 정보 * ⭐
│  │  ├─ 곡명 (텍스트)
│  │  ├─ 아티스트 (텍스트)
│  │  └─ 장르 (드롭다운)
│  │     옵션: 팝, 인디, 클래식, 록, 재즈, 힙합, 기타
│  └─ 메모 (textarea, 선택사항)
├─ 액션 버튼
│  ├─ 저장
│  └─ 취소
└─ 뒤로가기 버튼
```

### 6.3 여행 편집 (/edit/:id)

```
├─ 페이지 제목: "여행 편집"
├─ 폼 필드 (새 여행 추가와 동일)
│  └─ 기존 데이터 미리 채워짐
├─ 액션 버튼
│  ├─ 저장
│  ├─ 취소
│  └─ 삭제
└─ 뒤로가기 버튼
```

### 6.4 세계 지도 (/map) ⭐ 핵심 (추가!)

```
├─ 페이지 제목: "🗺️ 세계 지도"
├─ 통계 섹션
│  ├─ 방문한 국가 수
│  └─ 총 마커 개수
├─ 지도 영역
│  ├─ Leaflet 2D 지도
│  ├─ 여행지 마커 (장르별 색상)
│  └─ 줌/팬 컨트롤
├─ 마커 상호작용
│  ├─ 마커 클릭 → 팝업
│  │  ├─ 여행지명, 국가
│  │  ├─ 날짜
│  │  ├─ 음악 정보
│  │  └─ 장르
│  └─ 팝업 닫기
├─ 장르 범례
│  └─ 7개 장르의 색상 설명
└─ 뒤로가기 버튼
```

### 6.5 챗봇 (/chatbot) ⭐ 핵심

```
├─ 페이지 제목: "🎵 음악 기반 여행 챗봇"
├─ 채팅 영역
│  ├─ 시스템 메시지 (인사말)
│  │  "안녕하세요! 좋아하는 음악을 알려주시면 여행지를 추천해드릴게요."

│     ├─ 음악과의 연관성 설명
│     └─ 💡 과거 기록 참고
├─ 입력 폼
│  ├─ 텍스트 입력란
│  │  placeholder: "예: 잔잔한 인디 음악 좋아해, 어디 갈까?"
│  └─ 전송 버튼
```

## 9. 상호작용 패턴

### 9.1 일반 패턴

```
패턴 1: 여행 + 음악 저장 ⭐
[폼 입력] → [저장] → [로딩] → [Supabase에 저장]
├─ 1. songs 테이블에 음악 저장
└─ 2. trips 테이블에 여행 저장 (song_id 참조)
    ↓
[성공 메시지] → [대시보드로 리다이렉트]

패턴 2: 음악 기반 챗봇 ⭐
[질문 입력] → [전송] → [로딩]
    ↓
[LLM API 호출]
├─ 시스템 프롬프트 (음악-여행 연결 전문가)
├─ 사용자 질문
└─ 컨텍스트 (과거 여행 기록 + 음악 장르)
    ↓
[응답 표시]

패턴 3: 장르별 통계 확인 ⭐
[대시보드 로드] → [Supabase 조회]
    ↓
[장르별 카운트 계산]
    ↓
[카드/차트로 시각화]
```

### 9.2 피드백

```

- "여행이 저장되었습니다! 🎵" (토스트, 3초)

```

## 11. 마무리

이 간소화된 IA는 **음악과 여행의 연결 + 다양한 통계 시각화**를 핵심으로 합니다.

**페이지**:
- ✅ 대시보드 (/) - 장르별, 대륙별, 카테고리별 통계 + 여행 목록 ⭐ 강화
- ✅ 새 여행 추가 (/add) - 음악 + 좌표 + 대륙 + 카테고리 입력 ⭐ 강화
- ✅ 여행 편집 (/edit/:id)
- ✅ 세계 지도 (/map) - 2D, 장르별 색상 마커 ⭐
- ✅ 챗봇 (/chatbot) - 음악 기반 추천 ⭐

**핵심 유지 및 강화**:
- ✅ 여행마다 음악 정보 저장
- ✅ 세계 지도 시각화 (2D, 장르별 색상)
- ✅ 다양한 통계 시각화 ⭐ 강화
  - 음악 장르별 통계
  - 대륙별 여행 통계 (새로 추가)
  - 여행 카테고리별 통계 (새로 추가)
- ✅ 음악 기반 챗봇 추천

**간소화**:
- ❌ 로그인 (단일 사용자)
- ❌ 3D 지구본 (2D만)
- ❌ 지도 필터링 (시간 부족)
- ❌ 사진 업로드

**구현 시간**: 3시간
